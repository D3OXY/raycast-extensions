# Changelog

## [Initial Release] - {PR_MERGE_DATE}

- Grid UI with 6-column emoji-picker-style layout
- Quick links with URL, file path, and folder support
- Nested containers (any quick link can be a container)
- Multi-word search across all items including nested ones
- Frecency-based sorting (most used floats to top, alphabetical tiebreaker)
- Brand SVG icons (3000+ via Simple Icons) with color picker
- Auto high-res favicons via icon.horse
- Move quick links between containers and root
- Import/export via clipboard as JSON
- Launch argument for searching from Raycast global search
- Keyboard shortcuts for all actions
